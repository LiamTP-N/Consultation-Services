# Security Policy

## Supported Versions
We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
If you discover a potential security vulnerability, please do not open a public issue. Instead, please report it via the GitHub Security Advisory feature or email LiamTPearson@gmail.com.

Once a report is received, we will:
* Acknowledge the report within 48 hours.
* Provide an estimated timeline for a fix.
* Notify users once a patch is available.
