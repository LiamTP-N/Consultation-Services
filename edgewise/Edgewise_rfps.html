Filter Criteria - what gets shown and what gets dropped

Click to expand. Editing locked until you click "Unlock editing".

▸

How the filter works. Every tender pulled from the source portals is checked against three rules in order:

  1. Hard rejects. If the title or description contains any EXCLUDE keyword, the tender is dropped.
  2. Marine keyword match. If the tender contains any MARINE INCLUDE keyword, it is kept.
  3. CPV code with marine context. If the tender carries a tracked Canadian procurement code AND contains at least one MARINE CONTEXT word, it is kept.

Marine include keywords

Tenders matching any of these are kept (subject to the EXCLUDE list).

Canadian procurement category (CPV / UNSPSC) codes

Tenders carrying these codes are auto-included only if they also contain a marine context word.

Marine context words

Gate words used only with the CPV-code path. A CPV-coded tender must also contain at least one of these.

Exclude keywords

Hard-reject list. Any of these in the title or description drops the tender immediately.

How edits work

  • Click "Unlock editing", enter the passphrase. The x on each pill becomes clickable.
  • Click any x to remove that keyword. Type into an input box and press Enter to add.
  • Click "Save changes" - the new list is committed to the repo. Click "Run scrape now" at the top to apply immediately, otherwise the next daily scrape uses it.
