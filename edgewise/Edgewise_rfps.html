Filter Criteria - what gets shown and what gets dropped

Click to expand the full inclusion and exclusion keyword lists.

▸

How the filter works. Every tender pulled from the 21 source portals is checked against three rules in order:

  1. Hard rejects. If the title or description contains any EXCLUDE keyword, the tender is dropped.
  2. Marine keyword match. If the tender contains any MARINE INCLUDE keyword, it is kept.
  3. CPV code with marine context. If the tender carries a tracked Canadian procurement code AND contains at least one MARINE CONTEXT word, it is kept.

Otherwise, it is dropped. Some short acronyms (mmo, eem, eia, pam, sbo, pso, mmso) only match as standalone words to prevent false matches inside longer words like "engagement" or "agreement".
